# UI Hierarchy Model

## Shared/_Sites Structure

`Shared/_Sites/` provides the public website UI hierarchy:

- **Components/** - Reusable UI elements (Background)
- **Layouts/** - Page layout definitions
- **Nav/** - Navigation (MainPages, Section)
- **Pages/** - Page definitions with _Core base classes
- **Sections/** - Content sections (About, Agenda, Contact, Dish, Gallery, Welcome, Widgets)

Sections compose from reusable components, following wrapper + base pattern.
