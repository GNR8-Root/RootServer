### _Core Components
- CP_Dropdown, CP_DropdownGroup
- CP_HeadContentWebflow
- TabSimpleIcon

### Airtable Plugin Components
- C_TableAir - Table display
- C_RowSingle - Single row display
- C_Rows - Multiple rows
- C_Bookmark - Bookmark display

Components compose lower layers (displays, fields, pointers, actions) into cohesive UI units.
