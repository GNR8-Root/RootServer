
# Navigation & Information Architecture

---

## Breadcrumbs + Macro Stack

- Breadcrumbs represent the navigation stack
- Clicking a breadcrumb returns to that graph level
- Entering a macro always:
    - Saves layout silently
    - Transitions (fade)
    - Focuses the child root node (or saved camera)

**Example breadcrumb**:
```
Root Graph › AI Controller › Attack Behavior
```

## Global Actions Placement

- **Reload**: Near sync state (same mental model)
- **Validate**: Near edit actions
- **User/settings**: Far right
