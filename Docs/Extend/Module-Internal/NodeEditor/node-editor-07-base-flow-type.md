# Connection & Flow Types

Connections define meaning, not just order.

---

Supported flow concepts:
- Data Flow (value passing)
- Control Flow (execution order)
- Dependency Flow (execution prerequisites)
- Reference Flow (structure linking)

Each connection type enforces its own validation rules.
