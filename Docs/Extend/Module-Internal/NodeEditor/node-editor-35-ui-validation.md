# Validation UX

---

## When Validation Runs

- After edge creation
- After type changes
- Manual "Validate" button
- After applying remote updates

---

## How Validation Appears

- Node highlight ring (red/amber)
- Badge ⚠ on node corner
- Inspector "Validation" section lists items
- Canvas corner summary chip: `2 Errors · 1 Warning`

---

## Example Validation Entries

- **MISSING_ROOT** - Graph must have exactly one Root node
- **TYPE_MISMATCH** - Cannot connect object → number
