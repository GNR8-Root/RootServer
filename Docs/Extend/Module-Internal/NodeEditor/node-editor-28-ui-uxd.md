# Design System & User Experience

---

## Document Overview

This document defines the complete **UI + UX design** for a physics-based behavior tree editor built with **Blazor** (state + persistence), **Matter.js** (physics), and **p5.js** (rendering). It specifies the design system, layout, component behaviors, interaction flows, motion rules, accessibility, and UI examples.

### Design Principles

- **Clarity** - Strong hierarchy, low noise, predictable controls
- **Physicality** - Motion feels natural; physics supports intent, never fights it
- **Efficiency** - Keyboard shortcuts + fast repeated actions
- **Forgiveness** - Reversible actions, clear validation + recovery paths
- **Delight** - Subtle polish (glow, easing, micro-feedback) without distraction
