# Pointers Layer (03_Pointers)

## Pointer_Service

Central selection state service in Shared/_Core/00_Core/Events/Pointer_Service.cs

Tracks:
- Current workspace
- Current table
- Current row
- Current field

## Pointer Components

Airtable plugin implements pointer components:
- P_Table, P_Row, P_Field (selection)
- P_Workspace, P_Bookmarks (navigation)
- P_FieldsSelector, P_SelectorDropdown (UI)

Help pointers in _Core:
- HP_CurrentItem, HP_CurrentPath, HP_Structure
