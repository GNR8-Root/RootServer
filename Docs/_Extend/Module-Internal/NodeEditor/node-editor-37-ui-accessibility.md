
# Accessibility & Ergonomics

---

- **Port magnet radius**: Invisible 16–20px to improve targeting
- **Minimum hit targets**: 32px for toolbar buttons
- **Reduced motion mode**:
    - Disable momentum pan
    - Reduce glow pulses
- **Keyboard shortcuts (MVP)**:
    - `ESC`: Cancel / clear
    - `DEL`: Delete
    - `Ctrl/Cmd + F`: Focus search (phase 2)
    - `Alt + Left`: Back (macro)
